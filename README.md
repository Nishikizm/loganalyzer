# loganalyzer

Javaを用いたアプリ開発の学習を目的として、AIに出題させた課題に基づき作成しました。

## 概要

アプリのログファイル（UTF-8、1行1レコード）から、  
*level* が **'ERROR'** である行のみ抽出し、*module* 別に件数を集計する。

## 要件

- ログファイルのファイルパスをコマンドライン引数で受け取る
- *module* 別の **'ERROR'** 件数を昇順で表示
- 不正レコード（フィールド不足等）はスキップし、スキップ件数を表示
- 最後に総エラー数を表示

#### レコードの形式：　
```
timestamp　level　module　message
```
※区切りはスペース／タブの連続を想定  
※*message* は空白を含む文字列を想定

## ブランチ構成

- **main**：単一責任の原則に基づき、リファクタリングでクラスを分割（parser / analyzer / formatter）した実装
- **before-refactoring**：動作する成果物として最初に作成した単一クラスの実装
